# Processed Data – 100-ns DUSP15 Simulation

This directory contains processed time-series data generated from the 100-ns all-atom MD simulation of DUSP15 described in Section 2.6.2.

Included analyses:
- Global backbone RMSD
- Motif-specific RMSD (residues 75–95)
- Per-residue Cα RMSF
- Radius of gyration
- Motif SASA
- Intramolecular contact counts
- Minimum interatomic distances
- Hydration metrics
